# Hasta Kayıt Programı (C# WinForms)

## Açıklama
Bu proje, hastane için hasta kayıtlarını tutmak amacıyla geliştirilmiş bir **C# WinForms uygulamasıdır**.  
Hasta ekleme, silme, güncelleme gibi temel işlevleri içerir.  

## Özellikler
- Kullanıcı dostu arayüz
- SQL veritabanı bağlantısı
- Hasta bilgilerini kaydetme/güncelleme/silme
- Hata kontrolü

## Çalıştırma
1. Visual Studio’da projeyi açın.
2. `WinFormsApp3.sln` dosyasını çalıştırın.
3. Gerekirse veritabanı bağlantı ayarlarını güncelleyin.
